üì§ Send Configuration

  • Select a config file and protocol (SSH or Telnet).
  • Set a cooldown delay (in ms) between commands (e.g., 1000).
  • Click **Send Config** to apply it.
  • Or click **Test Send Config** to simulate the process without changes.
  • Use **Save Variables Only** to store variables without pushing configs.
